# Summary

The Extera Design Project Manager API offers the following classes:

  • Auth: responsible with authenticating the user
  • Project: provides means to access project level functionality.
  • Study: provides means to access study level functionality
  • Case: provides means to access case level functionality
  • Parameter: provides means to access parameter level functionality
