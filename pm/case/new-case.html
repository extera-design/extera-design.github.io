# The NewCase interface

The NewCase interface represents the data to be provided when a requesting to create a new case

interface NewCase {
  name: string,
  params: Array<string>
}

# NewCase fields

name: string

The name of the new case

params: Array<ParamValue>

Array of parameter values. Only parameters that are members of one the parent's supported parameter groups will be retained. If the parent's parameter groups contain more parameters than what is included in the params list, the missing parameters are added with their default value.

# Used by

  • Add new case to project
  • Add multiple new cases to project
  • Add new study to project
  • Add multiple new cases to study

# Example

{
  "name": "Case #1",
  "params": [
    { "id": "EDP$Alpha", "value": 2 }
  ],
}
