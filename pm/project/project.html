# The Project interface

The Project interface represents the data returned by the Project API that represents a project.

interface Project {
  _id: ObjectId,
  children: Array<ObjectId>,
  params: Array<string>,
  name: string,
  owner: ObjectId,
  lastAccessed: Date,
}

# Project fields

_id: ObjectId>

The unique ID of the project.

children: Array<ObjectId>

Array containing the list of IDs for this project's child cases and studies.

params: Array<string>

Array containing the list of IDs for this project's parameter groups.

name: string

The name of the project.

owner: ObjectID

The unique ID of the user that created this project.

lastAccessed: Date

The ISO 8601 (opens new window) date/time representation of the time when the project was created or last opened.
