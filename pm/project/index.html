# Project APIs

The Project APIs contain endpoints that allow access to the Project Manager's project. All APIs requests must use the Authorization header with the Bearer schema and provided an access token obtained from the Login API.

The following APIs are available:

  • Create new project
  • Get project
  • Get all projects
  • Open project
  • Update project
  • Create new case in project
  • Create multiple new cases in project
  • Create new study in project
