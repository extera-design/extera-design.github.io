# Login

API information
URL localhost:3600/auth
Method POST
Request body 
{
  "email": "login@email.com",
  "password": "password"
}
Response body 
{
  "accessToken": "eyJhbG ... VlEZ5ew9I",
  "refreshToken": "VTBRKz ... M0E9PQ=="
}

Request body parameters

email

The e-mail address that is the login for the user.

password

The user's password

Response body parameters

accessToken

The access token for the logged in user. This token will have to be passed in to every other API in order for the API to execute.

refreshToken

Not used for now. Future use will be to allow renewal of expired access tokens.
